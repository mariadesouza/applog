# applog
package to support leveled logs in Go


## Methods

-  Error(a ...interface{})

-  Errorf(format string, a ...interface{})

-  Info(a ...interface{})

-  Infof(format string, a ...interface{})

-  Trace(a ...interface{})

-  Tracef(format string, a ...interface{})

-  Warn(a ...interface{})

-  Warnf(format string, a ...interface{})
